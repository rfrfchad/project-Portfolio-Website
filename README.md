# Portfolio Website

This project is a modern portfolio website designed to showcase personal information, resume details, portfolio projects, and contact information through a clean and responsive interface.

## Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript

## Deployment

The website was deployed using:

* InfinityFree
* ProFreeHost

## Features

* Responsive portfolio design
* Portfolio and resume sections
* Contact section
* Dynamic content using PHP and MySQL
* Admin panel for website management
